# B 站视频采集分析方案

_UP 主历史视频数据采集与分析完整方案_

**创建日期：** 2026-03-23  
**版本：** 1.0

---

## 📋 方案概述

本方案用于采集 B 站指定 UP 主一年内的视频数据，并进行多维度分析，产出数据报告。

### 适用场景

- UP 主成长分析
- 竞品账号分析
- 内容策略优化
- 粉丝偏好研究

## 🚀 快速开始

### 步骤 1：获取 UP 主 UID

1. 打开 B 站 UP 主主页
2. URL 格式：`https://space.bilibili.com/[UID]`
3. 复制 UID 数字

### 步骤 2：配置脚本

编辑 `bilibili-collect.py`：

```python
UP_UID = "12345678"  # 替换为目标 UP 主 UID
```

### 步骤 3：执行采集

```bash
python bilibili-collect.py
```

**输出：**
- `bilibili_data/[UP 主名]_videos_YYYYMMDD_YYYYMMDD.csv`

### 步骤 4：执行分析

编辑 `bilibili-analyze.py` 中的文件名，然后：

```bash
python bilibili-analyze.py
```

**输出：**
- `bilibili_analysis/analysis_report_YYYYMMDD_HHMM.md`
- `bilibili_analysis/analysis_data_YYYYMMDD_HHMM.xlsx`

## 📈 分析维度

### 1. 基础统计

| 指标 | 说明 |
|------|------|
| 视频总数 | 统计周期内发布视频数量 |
| 总播放量 | 所有视频播放量总和 |
| 总互动数 | 点赞 + 硬币 + 收藏 + 分享 |
| 平均播放 | 单视频平均播放量 |
| 互动率 | 点赞数/播放量 × 100% |
| 三连率 | (点赞 + 硬币 + 收藏)/播放量 × 100% |

---

### 2. 趋势分析

| 维度 | 说明 |
|------|------|
| 月度趋势 | 每月视频数、播放量、互动数变化 |
| 播放趋势 | 识别上升/下降趋势 |
| 粉丝增长 | 根据播放量推测粉丝增长 |

---

### 3. 内容分析

| 维度 | 说明 |
|------|------|
| 分区分布 | 各分区视频数量及表现 |
| 时长分析 | 视频时长与播放量关系 |
| 标题分析 | 高频词、标题长度分析 |
| 封面分析 | （需额外采集） |

---

### 4. 时间分析

| 维度 | 说明 |
|------|------|
| 星期分布 | 周几发布视频最多/表现最好 |
| 小时分布 | 几点发布视频效果最佳 |
| 发布频率 | 平均几天发布一个视频 |

---

### 5. TOP 榜单

| 榜单 | 说明 |
|------|------|
| 播放 TOP10 | 播放量最高的 10 个视频 |
| 点赞 TOP10 | 点赞数最高的 10 个视频 |
| 弹幕 TOP10 | 弹幕数最高的 10 个视频 |
| 互动 TOP10 | 综合互动最高的 10 个视频 |

## ⚠️ 注意事项

### API 限制

| 限制 | 说明 |
|------|------|
| 请求频率 | 建议间隔 0.5-1 秒 |
| 每日限额 | 未登录约 1000 次/天 |
| 登录限制 | 登录后可提高限额 |

### 反爬措施

- 添加 User-Agent
- 控制请求频率
- 使用 Cookie 登录态
- 避免高频采集

### 数据准确性

- 部分实时数据（如当前播放量）可能与采集时有差异
- 建议定期更新数据
