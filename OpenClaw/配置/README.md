# OpenClaw 配置中心

所有配置文件和使用说明

## 文件列表

| 文件 | 用途 | 说明 |
|------|------|------|
| CONFIG.md | 配置说明总览 | 所有配置文件的详细文档 |
| ip-subagent.txt | 子节点服务器列表 | 部署脚本使用的服务器清单 |
| lessons.md | 踩坑记录 | 部署和运维问题总结 |

## 快速开始

1. 配置服务器列表 - 编辑 ip-subagent.txt
2. 执行部署 - ./deploy-subagent.sh
3. 查看说明 - cat CONFIG.md

## 安全提示

ip-subagent.txt 包含敏感信息：
- 不要提交到公共 Git 仓库
- 部署后建议删除或加密
- 可使用 .gitignore 忽略
