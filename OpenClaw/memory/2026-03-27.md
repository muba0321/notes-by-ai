### 🔧 修复步骤

1. **生成 SSL 证书（带 SANs）**
```bash
openssl req -x509 -nodes -days 3650 -newkey rsa:4096 \
  -keyout /data/harbor/ssl/server.key \
  -out /data/harbor/ssl/server.crt \
  -subj '/CN=harbor.mubai.top/O=Harbor/C=CN' \
  -addext 'subjectAltName=DNS:harbor.mubai.top,DNS:*.harbor.mubai.top,IP:38.246.245.39'
```

2. **配置 harbor.yml**
```yaml
hostname: harbor.mubai.top
https:
  port: 8443
  certificate: /data/harbor/ssl/server.crt
  private_key: /data/harbor/ssl/server.key
```

3. **修复 DNS 解析**
```bash
# 主节点和子节点 1 都添加
echo '38.246.245.39 harbor.mubai.top' >> /etc/hosts
```

4. **配置 Docker 信任**
```bash
mkdir -p /etc/docker/certs.d/harbor.mubai.top:8443
cp /data/harbor/ssl/server.crt /etc/docker/certs.d/harbor.mubai.top:8443/ca.crt
```

5. **创建项目**
```bash
curl -X POST "https://harbor.mubai.top:8443/api/v2.0/projects" \
  -H "Authorization: Basic YWRtaW46aHVhbnhpbjAzMjE=" \
  -d '{"project_name":"cmdb","metadata":{"public":"true"}}'
```

6. **推送镜像**
