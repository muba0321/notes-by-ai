# OpenClaw 部署文档

OpenClaw 服务端和子节点部署指南

## 目录结构

- 服务端/ - 主节点部署脚本和文档
- 子节点/ - 子节点部署脚本和文档
- 配置/ - 配置文件和说明

## 快速开始

### 部署服务端

cd 服务端
./deploy_openclaw_server.sh

### 部署子节点

1. 编辑配置/ip-subagent.txt
2. ./deploy-subagent.sh
