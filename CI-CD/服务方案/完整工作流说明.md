### 阶段 2：代码生成

**位置：** coder-api 仓库

**步骤：**
```bash
# 1. 创建代码目录
mkdir -p /data/openclaw-dist/code/coder-api/[项目名]

# 2. 调用 aliyun-coder 生成代码
curl -X POST "https://coding.dashscope.aliyuncs.com/v1/chat/completions" \
  -H "Authorization: Bearer sk-sp-7e6f845b069f486d9b18aa8366579f1e" \
  -H "Content-Type: application/json" \
  -d "{
    \"model\": \"qwen3.5-plus\",
    \"messages\": [{
      \"role\": \"user\",
      \"content\": \"根据 PRD 生成代码：[PRD 内容]\"
    }]
  }"

# 3. 创建依赖和 Dockerfile
cat > requirements.txt << 'EOF'
fastapi==0.109.0
uvicorn==0.27.0
EOF

cat > Dockerfile << 'EOF'
FROM python:3.11-slim
WORKDIR /app
COPY requirements.txt .
RUN pip install -r requirements.txt
COPY . .
EXPOSE 8000
CMD ["python", "main.py"]
EOF

# 4. Git 提交
cd /data/openclaw-dist/code/coder-api/[项目名]
git add -A
git commit -m "feat: [项目名] 代码生成"
git push origin main
```

### 阶段 3：容器部署

**步骤：**
```bash

# 2. 启动容器

  --name [项目名] \
  -p 8000:8000 \
  --restart unless-stopped \

# 3. 验证

```
